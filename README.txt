Eurobot 2015 Hauptsteuerung
===========================

This software has two parts. One runs on the BeagleBone and controls the robot. The other is running on the laptop.

To start the programm on the BeagleBone ::
    Python3 `PATH TO PROJECT`/eurobot/hauptsteuerung_main.py

To start the programm on the laptop ::
    Python3 `PATH TO PROJECT`/eurobot/laptop_main.py

The full documentation for this software is in /docs/_build/html.
